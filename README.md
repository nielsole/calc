# JustCalc simple CLI calculator

A lot of times I just want to calculate something quickly.
I hate having to resort to Google for calculating basic things.

## Features

* subtraction, addition
* division, multiplication
* power, modulo
* parenthesis
* PI

## Usage

```
$ ./mylang 
5*2^0.5*PI
22.21441469
4+2
6
-2-2
-4
^C
```
